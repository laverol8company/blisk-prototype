# BLISK — прототип сайта детейлинг-студии

Премиальный one-pager для авто-детейлинга (украинский рынок). Тёмная тема «Хром и лёд»: графит + хром + холодный электрик-голубой акцент. Контент на украинском.

> `BLISK` (укр. «блиск» = глянец) — рабочее имя бренда. Меняется за минуту, см. ниже.

## Как запустить

Три способа:

1. **Просто открыть** `index.html` двойным кликом в браузере.
2. **Локальный сервер** (рекомендую, чтобы корректно грузились пути):
   ```powershell
   node .claude/serve.js
   # откроется на http://localhost:4321
   ```
3. **Панель Preview** в Claude Code (конфиг уже лежит в `.claude/launch.json`).

Зависимости подгружаются с CDN (шрифты, GSAP, иконки Phosphor, логотипы марок, фото-плейсхолдеры), поэтому для первого показа нужен интернет.

## Структура

```
index.html      разметка и весь украинский контент
styles.css      дизайн-система + все секции (хром, отражения, бенто, слайдер)
script.js       вода на canvas, GSAP-скролл, калькулятор, драг-галерея, магнитные кнопки
.claude/        serve.js (мини-сервер) + launch.json (конфиг превью)
assets/         сюда положить реальные фото (см. ниже)
```

## Фишки

- **Гидрофобный курсор** в герое: вода собирается в капли и скатывается по кузову (canvas).
- **Слайдер «До / После»** с имитацией прохода полировки (риски → зеркало).
- **Пиннинг-скролл процесса**: 6 этапов едут горизонтально (GSAP ScrollTrigger).
- **Калькулятор пакета**: класс авто × пакет + допы → живая вилка цены и срок.
- **Драг-галерея** работ с инерцией.
- Магнитные CTA, счётчики, reveal-анимации, бренд-маркизы.
- Полный respect к `prefers-reduced-motion` и адаптив до мобильного.

## Что поменять под клиента

| Что | Где |
|---|---|
| **Имя бренда** `BLISK` | `index.html` (шапка, футер, `<title>`) |
| **Акцентный цвет** | `styles.css` → `:root` → `--accent`, `--accent-2` |
| **Шрифты** | `index.html` (ссылка Google Fonts) + `styles.css` → `--font-display/body` |
| **Цены и пакеты** | `index.html`, секция `#calc`: атрибуты `data-min/data-max/data-days` (пакеты), `data-mult` (класс), `data-add` (допы) |
| **Телефон, адрес, соцсети** | `index.html`, секция `#contact` и футер |
| **Тексты, услуги, отзывы** | `index.html` (всё инлайн) |

## Реальные фото

Сейчас стоят плейсхолдеры (picsum) через `onerror`-фолбэк. Положите файлы в `assets/` с такими именами, и сайт подхватит их автоматически:

- `assets/car-hero.jpg` — герой, глянцевое авто в студии, ~1800×1100
- `assets/result.jpg` — кузов для слайдера «до/после», ~1600×900
- `assets/interior.jpg` — салон, ~800×700
- `assets/work-1.jpg` … `work-6.jpg` — работы в галерею, ~700×520

Лучший эффект — тёмные студийные кадры с отражениями.
